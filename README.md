# gazexpress

"Bringing gas delivery into the digital age in Cameroon."

GazExpress is a web application that enables users to order cooking gas online in Douala, Cameroon. It connects customers with gas vendors through a simple and user-friendly interface.

The platform allows customers to place orders, track delivery status, and receive updates via WhatsApp. Administrators can manage orders, assign vendors, and update delivery progress in real time using Firebase.

Built with React and Firebase, GazExpress provides a fast, scalable, and modern solution to improve gas distribution services.
